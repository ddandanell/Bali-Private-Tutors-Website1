# Favicon Setup Instructions

This project requires favicon files to be added to the `/public` directory:

Required files:
- favicon.ico (16x16 and 32x32)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- og-image.jpg (1200x630 for Open Graph)
- twitter-card.jpg (1200x600 for Twitter)
- logo.png (for structured data)

You can generate these using online tools like:
- https://realfavicongenerator.net/
- https://favicon.io/

Upload your logo/brand image and download the generated favicon package.

For now, the site will use browser default favicons until these files are added.
